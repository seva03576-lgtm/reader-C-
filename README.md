# Многопоточный парсер логов на C++17

Консольная утилита для быстрой фильтрации тяжелых файлов логов (от нескольких гигабайт) с использованием пула потоков.

Что Умеет:
* Поиск без лишнего копирования памяти через `std::string_view`.
* Собственный ThreadPool для параллельной обработки кусков файла.
* Асинхронный ProgressBar в отдельном потоке (вывод скорости в МБ/с и счетчика совпадений).
* Кроссплатформенность (Windows / Linux / macOS).

## Сборка (Release)
```bash
mkdir build && cd build
cmake -DCMAKE_BUILD_TYPE=Release ..
cmake --build .
```
Использование
./log_parser <путь_к_файлу> <запрос> [выходной_файл] [потоки] [кусок_мб]
Пример:
./log_parser server.log "ERROR" errors.log 8 64
